

## 快捷键

+ 1.md设置文件目录 快捷键：`⌘N` 或者 `^⏎` 然后选则 table of contents
+ 2.查看源码时我们一般要看类关系图：快捷键：`⌘⌥⇧U` 。 如果要看接口实现在 `⌘⌥B`，继承：`⌘⌥P` 。 参考：https://www.jianshu.com/p/bb0602619461
+ 3.`^H`  文件继承关系，(structure,该效果比较好) 。 `^⌥H` 方法被调用情况。
