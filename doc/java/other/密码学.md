# 密码学

##### 1.算法种类
+ 对称加密：加密解密使用同一个密钥
+ 非对称加密：加密解密使用两个不同的密钥，一个公钥，一个私钥。公钥与私钥一对，公钥加密数据，必须用私钥解密
+ 哈希算法 ：主要用于hash值计算


##### 2.非对称加密举例
|    |SM2  |RSA |
|:---|:---:|---:|
|算法结构    |椭圆曲线（ECC）|基于可逆的模幂运算|
|计算复杂度  |指数级        |压指数级 |
|存储空间    |256bit       |2048-4096bit |
|密钥生成速度 |比RSA快百倍以上|慢|
|加密解密速度 |较快          |一般 |
